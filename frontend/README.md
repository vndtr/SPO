# Образовательная платформа для совместного чтения

Веб-приложение для совместного чтения книг с возможностью создания аннотаций (заметок и цитат), организации сессий и взаимодействия между учителем и учениками.

## О проекте

Данный проект представляет собой прототип образовательной системы, предназначенной для совместного чтения учебной литературы. Основная цель — обеспечить возможность взаимодействия преподавателей и учеников непосредственно в тексте через создание аннотаций и обратную связь.

**Ключевая особенность:** приватная обратная связь преподавателя конкретному ученику и возможность 
создания как личных, так и публичных заметок.

## Функциональные возможности

### Личное чтение
- Чтение текста произведения (А.П. Чехов "Тоска")
- Создание цитат с выделением цветом (жёлтый, зелёный, синий, розовый)
- Создание заметок с комментариями и цветным подчёркиванием
-  Автоматическое сохранение всех аннотаций в localStorage
-  Прокрутка к месту аннотации по клику на карточку
- Удаление аннотаций с мгновенным снятием подсветки

###  Совместное чтение (сессии)
-  Ролевая модель (учитель/ученики)
-  Публичные и приватные заметки
-  Возможность отвечать на чужие заметки
-  Древовидная структура ответов
-  Фильтрация аннотаций по типу и автору
- Просмотр участников сессии

###  Профиль пользователя
-  Редактирование личных данных
- Сохранение изменений в localStorage
-  Отмена редактирования

## Технологический стек

- **React 18** — библиотека для построения пользовательского интерфейса
- **React Router 6** — маршрутизация
- **Tailwind CSS** — стилизация компонентов
- **LocalStorage** — хранение данных (прототип без серверной части)

## Установка и запуск

### Требования
- Node.js 16+ 
- npm или yarn
